# 📖 Glossary of Key Terms

> 🎓 **Course:** NOC: Introduction to Industry 4.0 and Industrial Internet of Things
> 👨‍🏫 **Instructor:** Prof. Sudip Misra, IIT Kharagpur
> 📌 Terms are organized by topic for easy reference.

---

## 🏭 Industry 4.0

| Term | Full Form | Definition |
|------|-----------|------------|
| **I4.0** | Industry 4.0 | The fourth industrial revolution — integration of digital, physical and biological systems in manufacturing |
| **CPS** | Cyber-Physical Systems | Systems that integrate computation, networking and physical processes in real time |
| **PLM** | Product Lifecycle Management | Managing a product's lifecycle from design to disposal using digital tools |
| **AR** | Augmented Reality | Overlay of digital information onto the real physical world |
| **VR** | Virtual Reality | Fully immersive computer-generated environment simulating real-world scenarios |
| **LEAN** | Lean Production System | Manufacturing philosophy focused on minimizing waste while maximizing productivity |
| **Digital Twin** | — | A virtual replica of a physical system, process, or product used for simulation and monitoring |
| **Smart Factory** | — | A highly digitized and connected manufacturing facility that uses automation and data exchange |
| **M2M** | Machine-to-Machine | Direct communication between devices without human intervention |
| **AMR** | Autonomous Mobile Robots | Self-guided robots used in industrial environments for material handling |

---

## 🌐 Internet of Things (IoT)

| Term | Full Form | Definition |
|------|-----------|------------|
| **IoT** | Internet of Things | Network of physical objects embedded with sensors, software and connectivity to exchange data |
| **IIoT** | Industrial Internet of Things | Application of IoT specifically in industrial and manufacturing environments |
| **Actuator** | — | A device that receives a signal and performs a physical action (e.g., motor, valve) |
| **Sensor** | — | A device that detects and measures physical quantities and converts them to signals |
| **Gateway** | — | A device that bridges different networks/protocols — connects field devices to the cloud |
| **Edge Device** | — | Hardware that processes data at or near the source rather than sending it to the cloud |
| **Node** | — | Any device connected to a network — sensor, actuator, gateway, or controller |
| **Firmware** | — | Low-level software embedded in hardware devices that controls their core operations |
| **Middleware** | — | Software that connects different applications or services in an IoT system |
| **RFID** | Radio Frequency Identification | Uses electromagnetic fields to automatically identify and track tags attached to objects |

---

## 📡 Communication & Networking

| Term | Full Form | Definition |
|------|-----------|------------|
| **MQTT** | Message Queuing Telemetry Transport | Lightweight publish-subscribe messaging protocol designed for IoT devices |
| **AMQP** | Advanced Message Queuing Protocol | Open standard messaging protocol for message-oriented middleware |
| **CoAP** | Constrained Application Protocol | Lightweight web protocol for use with constrained nodes and networks in IoT |
| **HTTP** | HyperText Transfer Protocol | Standard protocol for communication over the web |
| **REST** | Representational State Transfer | Architectural style for designing networked applications using HTTP |
| **OPC-UA** | Open Platform Communications Unified Architecture | Industrial communication standard for secure and reliable data exchange |
| **TCP/IP** | Transmission Control Protocol / Internet Protocol | Foundational communication protocols for the internet |
| **Wi-Fi** | Wireless Fidelity | Wireless networking technology based on IEEE 802.11 standards |
| **Zigbee** | — | Low-power, low-data-rate wireless mesh network standard for IoT devices |
| **Z-Wave** | — | Wireless protocol designed for home automation with low power consumption |
| **LoRa** | Long Range | Low-power wide-area network (LPWAN) technology for long-range IoT communication |
| **LoRaWAN** | Long Range Wide Area Network | Network protocol built on LoRa for large-scale IoT deployments |
| **NB-IoT** | Narrowband IoT | Cellular network standard for low-power IoT devices over wide areas |
| **BLE** | Bluetooth Low Energy | Short-range wireless protocol optimized for low-power IoT devices |
| **6LoWPAN** | IPv6 over Low-Power Wireless Personal Area Networks | Standard to enable IPv6 packets over IEEE 802.15.4 networks |
| **WAN** | Wide Area Network | Network that spans large geographic areas |
| **LAN** | Local Area Network | Network confined to a small area like a building or campus |
| **PAN** | Personal Area Network | Short-range network for personal devices |

## ☁️ Cloud, Fog & Edge Computing

| Term | Full Form | Definition |
|------|-----------|------------|
| **Cloud Computing** | — | Delivery of computing services (storage, processing, networking) over the internet |
| **Fog Computing** | — | Extends cloud capabilities to the edge of the network, closer to data sources |
| **Edge Computing** | — | Processing data at or near the source device rather than in a centralized cloud |
| **SaaS** | Software as a Service | Cloud-hosted software delivered over the internet (e.g., Google Workspace) |
| **PaaS** | Platform as a Service | Cloud platform for developing and deploying applications (e.g., AWS Elastic Beanstalk) |
| **IaaS** | Infrastructure as a Service | Cloud-based infrastructure like virtual machines and storage (e.g., AWS EC2) |
| **Latency** | — | Time delay between data being sent and received — critical in real-time IIoT systems |
| **Bandwidth** | — | Maximum rate of data transfer across a network |
| **CDN** | Content Delivery Network | Distributed network of servers that delivers content based on user's geographic location |

## 🔐 Cybersecurity

| Term | Full Form | Definition |
|------|-----------|------------|
| **CIA Triad** | Confidentiality, Integrity, Availability | The three core principles of information security |
| **Firewall** | — | Network security system that monitors and controls incoming/outgoing traffic |
| **IDS** | Intrusion Detection System | Monitors network traffic for suspicious activity and alerts administrators |
| **IPS** | Intrusion Prevention System | Actively blocks detected threats in addition to monitoring |
| **VPN** | Virtual Private Network | Encrypted connection over the internet for secure remote access |
| **PKI** | Public Key Infrastructure | Framework for managing digital certificates and public-key encryption |
| **TLS** | Transport Layer Security | Cryptographic protocol that provides secure communication over a network |
| **DoS** | Denial of Service | Attack that overwhelms a system to make it unavailable to users |
| **DDoS** | Distributed Denial of Service | DoS attack launched from multiple compromised systems simultaneously |
| **SCADA** | Supervisory Control and Data Acquisition | Industrial control system for monitoring and controlling industrial processes |
| **PLC** | Programmable Logic Controller | Industrial digital computer used to automate electromechanical processes |
| **Zero-Day** | — | A software vulnerability unknown to the vendor, exploited before a patch is available |
| **Malware** | Malicious Software | Any software designed to disrupt, damage, or gain unauthorized access to systems |

## 📊 Data Analytics & Machine Learning

| Term | Full Form | Definition |
|------|-----------|------------|
| **Big Data** | — | Extremely large datasets that cannot be processed with traditional tools — defined by Volume, Velocity, Variety |
| **ML** | Machine Learning | Subset of AI where systems learn from data to improve performance without explicit programming |
| **AI** | Artificial Intelligence | Simulation of human intelligence processes by machines |
| **Deep Learning** | — | ML subset using multi-layered neural networks to learn complex patterns |
| **Hadoop** | — | Open-source framework for distributed storage and processing of large datasets |
| **MapReduce** | — | Programming model for processing large data sets in parallel across a cluster |
| **HDFS** | Hadoop Distributed File System | Distributed file system designed to run on commodity hardware for Big Data |
| **Supervised Learning** | — | ML approach where the model is trained on labelled data |
| **Unsupervised Learning** | — | ML approach where the model finds patterns in unlabelled data |
| **Regression** | — | Statistical method to model the relationship between variables for prediction |
| **Classification** | — | ML task of predicting which category an input belongs to |
| **Clustering** | — | Grouping similar data points together without predefined labels |
| **R** | — | Programming language and environment for statistical computing and graphics |
| **Julia** | — | High-performance programming language for numerical and scientific computing |
| **ETL** | Extract, Transform, Load | Process of extracting data from sources, transforming it, and loading into a destination |
| **KPI** | Key Performance Indicator | Measurable value showing how effectively an objective is being achieved |

## 🔀 Software Defined Networking (SDN)

| Term | Full Form | Definition |
|------|-----------|------------|
| **SDN** | Software Defined Networking | Network architecture that separates the control plane from the data plane |
| **NFV** | Network Function Virtualization | Virtualizing network services that traditionally ran on dedicated hardware |
| **Control Plane** | — | The part of a network that decides how traffic is routed |
| **Data Plane** | — | The part of a network that actually forwards traffic based on control plane decisions |
| **OpenFlow** | — | Communication protocol that gives access to the forwarding plane of a network switch |
| **SDN Controller** | — | Centralized software that manages flow control and network configuration |
| **Overlay Network** | — | Virtual network built on top of an existing physical network |

## 🏗️ Industrial Systems & Architecture

| Term | Full Form | Definition |
|------|-----------|------------|
| **SCADA** | Supervisory Control & Data Acquisition | System for remote monitoring and control of industrial infrastructure |
| **DCS** | Distributed Control System | Industrial control system with decentralized elements across the plant |
| **HMI** | Human-Machine Interface | Dashboard/interface that allows operators to interact with industrial systems |
| **MES** | Manufacturing Execution System | System that tracks and controls manufacturing operations in real time |
| **ERP** | Enterprise Resource Planning | Business process management software integrating core business functions |
| **OT** | Operational Technology | Hardware and software for monitoring/controlling physical devices and processes |
| **IT** | Information Technology | Use of computers and systems to store, retrieve and transmit data |
| **IT/OT Convergence** | — | Integration of traditional IT systems with industrial OT systems in I4.0 |
| **ISA-95** | — | International standard for integrating enterprise and control systems |
| **IEC 62443** | — | Series of standards for cybersecurity in industrial automation and control systems |

## 🚁 Application Domains

| Term | Full Form | Definition |
|------|-----------|------------|
| **UAV** | Unmanned Aerial Vehicle | A drone — aircraft without a human pilot onboard, used for industrial inspection |
| **AGV** | Automated Guided Vehicle | A vehicle that moves automatically along marked paths in a factory or warehouse |
| **CMMS** | Computerized Maintenance Management System | Software for managing maintenance operations in a facility |
| **AMM** | Automated Meter Management | System for remote reading and management of utility meters |
| **Cold Chain** | — | Temperature-controlled supply chain for perishable goods (e.g., food, pharma) |
| **Predictive Maintenance** | — | Using sensor data and ML to predict equipment failure before it occurs |
| **Condition Monitoring** | — | Continuous monitoring of equipment parameters to detect deterioration |
| **Quality Control** | — | Processes ensuring products meet defined standards before delivery |
